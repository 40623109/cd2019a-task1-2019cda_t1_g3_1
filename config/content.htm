隨身系統

藉由自行製作隨身系統，來了解隨身系統的運作方式，以便了解工具的使用與設定，並可以量身打造所需的隨身系統。

創建資料夾

創建一個放置可攜檔案的資料夾

並在裡面創建一個data資料夾用來連結y槽

SciTE

安裝SciTE至data資料夾中

使用windos的 full download版本

SciTE Download:https://www.scintilla.org/SciTEDownload.html

打開SciTE後

開啟設定檔案

利用ctrl+f找到code.page並改為下圖所示

安裝 python 進可攜系統

安裝時選取自訂義安裝

勾選所需要的項目

設定安裝位置

利用HxD二位元編輯器設定C:\p37\Scripts\pip.exe

使用ctrl+f功能尋找c:\p37並改為y:\p37

如紅字處

更改完成後就可以把整個p37資料夾移至data資料夾

start.bat

利用安裝的SciTE製作start.bat檔案

stop.bat

利用安裝的SciTE製作stop.bat檔案

在製作過程中想要快速了解兩個檔案之間的差異可以使用Kdiff3來快速比對出不一樣的地方。

ungit

使用圖形化介面讓使用者更加了解目前所在的分支與倉儲狀況

如何安裝

參考連結http://mde.tw/wcm2019/content/%E5%AF%A6%E7%BF%92%E9%A0%85%E7%9B%AE%E4%BA%8C.html

至上方連結處下載 nodejs_with_ungit.7z

下載完成後解壓縮到隨身系統資料夾的data中

並在start.bat中加入

set USERPROFILE=%Disk%:\home

set path8=%Disk%:\nodejs;%Disk%:\nodejs\appdata\roaming\npm;

%path8%;

如下圖

之後在中空白的SciTE中輸入

ungit --launchBrowser=0 --launchCommand "C:\PROGRA~2\Google\Chrome\Application\chrome.exe  %%U"

存檔至Data中並命名為kungit.bat
